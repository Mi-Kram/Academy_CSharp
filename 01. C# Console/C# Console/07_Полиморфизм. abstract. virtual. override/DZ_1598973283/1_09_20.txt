1. Разработать класс, который позволяет генерировать простые числа при помощи индексатора.
При обращении к индескатору переменной класса, возвращает простое число по порядку возрастания.

Пример:

PrimeNumber pr = new PrimeNumbers()
for(int i=0; i<10; i++)
{
	Console.WriteLine(pr[i]);
}

Вывод:
pr[0] - 2
pr[1] - 3
pr[2] - 5
pr[3] - 7
pr[4] - 11
pr[5] - 13
pr[6] - 17
pr[7] - 19
pr[8] - 23
pr[9] - 29

2. Разработать класс, который позволяет получать доступ к любому слову текстового файла 
по индексатору. Можно читать и менять слова. Слова отделены друг от друга пробелами и новыми
строками.

Пример:
FileWords words = new FileWords(@"c:\temp\text1.txt");
words[0] = "Пока";
for(int i=0; i<words.Length; i++)
{
	Console.WriteLine(words[i]);
}

Вывод:

Пока
мир
