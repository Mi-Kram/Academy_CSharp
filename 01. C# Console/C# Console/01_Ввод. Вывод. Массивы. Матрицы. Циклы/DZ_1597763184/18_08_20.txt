1. Пользователь вводит размеры двумерного массива и сам массив. Программа вычисляет среднее
арифметическое минимального и максимального элементов массива.

2. Пользователь вводит размер одномерного массива и сам массив. Программа выясняет, состоит ли
заданный массив только из простых чисел или нет.

3. Пользователь вводит размер одномерного массива и сам массив. Программа находит в массиве
2 максимальных числа за один проход массива.

Примеры:

1, 2, 3, 4, 5

Res: 5, 4

1, 2, 3, 5, 5
Res: 5, 5

9, 8, 7, 6, 5
Res: 9, 8

0, 0, 0, 0, 0
Res: 0, 0
