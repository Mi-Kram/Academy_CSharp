1. При помощи делегатов реализовать сортировку с использованием компараторов для сортировки
одномерных массивов в порядке возрастания и убывания.

Пример:
Vector v = new Vector();
v.Add(1, 12, 9, 4, 5);

// сортировка в порядке возрастания
v.Sort(ascend);

// сортировка в порядке убывания
v.Sort(descend);

2. Реализовать при помощи events. Сервер содержит внутри себя таймер, который посылает всем
подписчикам 1 раз в сек. сообщения от текущей дате и времени. Первый подписчик печатает время на
экран. Второй подписчик пишет время в файл .log. Третий подписчик выключает сервер через 30 сек
его работы.
