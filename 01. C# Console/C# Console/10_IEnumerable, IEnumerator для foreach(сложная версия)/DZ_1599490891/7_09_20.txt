1. В класс Matrix добавить возможность просмотра в цикле foreach хранимой в классе матрицы.
Матрица просматривается слева-направо и сверху-вниз. Использовать интерфейсы IEnumerable,
IEnumerator.

2. В класс PrimeNumbers добавить возможность просмотра в цикле foreach генерируемых простых чисел.
Использовать интерфейсы IEnumerable, IEnumerator.

Пример:

PrimeNumbers pr = new PrimeNumbers(100);

// вывод на экран 100 простых чисел
foreach(int num in pr)
{
	Console.WriteLine(num);
}
