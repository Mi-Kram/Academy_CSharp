1. ASP.NET MVC
Разработать сайт, который позволяет просматривать автомобили из коллекции автомобилей из памяти.
Модель автомобиля:
Car
Brand
Model
Price
Speed
Year
Type (Sedan, PickUp, Convertible, Roadster)
picture -  картинка машины

Дополнительные маршруты:
\Name\BMW - показать все BMW
\Year\1998 - показать все машины по году
\Speed\200 - показать машины со скоростью больше 200
\Price\10000 - показать машины дороже цены

2. Добавить возможность добавления, редактирования и просмотра машин (с загрузкой картинки авто на сервер)

3. Добавить валидацию полей форм для добавления и редактирования авто.
Использовать разные атрибуты, атрибут Remote, добавить custom validation

4. Добавить в проект Tag-helpers, которые добавляют на сайт paging для машин и добавить меню в виде Tag-helper

5. Добавить регистрацию, вход и выход пользователей на сайт. Информацию о пользователях хранить в файле
пользователей. При входе пользователя создавать для него сессию. У каждого пользователя может быть его список
машин. Добавить админскую учётку, которая позволяет видеть машины всех пользователей.

6. Разработать БД с автомобилями и таблицы в ней. Заполнить БД.
Подключить БД к проекту через Database First. На странице Index добавить сортировку для всех полей при щелчке
на столбце. Чётные и нечётные строки показывать с разным фоном. В таблице и в деталях показывать фото машины.
URL картинки хранится в БД. Добавить paging.

7. Реализовать все функции проекта при помощи подхода Code First, включая миграции (создание таблиц, заполнение)
